Hakkımda

Yazılıma uzun süredir merakım olduğu için kendimi front-end teknolojileri konusunda geliştirmeye başladım. The Odin Project sitesi üzerinden Front End Developer eğitimime devam ediyorum. Html, Css, Scss i aktif olarak kullanabiliyorum. Javascript konusunda ise kendime geliştirmeye devam ediyorum. Bu yaptığım mobil uyumlu sitede giriş seviyesinde Javascript kullanarak fonksiyonellik eklemeyi amaçladım. Sitenin projeler kısmı şuan için boş fakat ileryen günlerde css,js kullanarak yapacağım diğer statik siteleri ekleme amacındayım. Github hesabımda Javascript öğrenmeye çalışırken yaptığım ufak projeler mevcut. MySQL ile ise aktif olarak bir database üzerinde çalışmadım fakat daha önce üniversite vasıtasıyla temel seviyede eğitimini aldım. Linux üzerinden çalıştığım için terminal kodlarına ve git/github entegrasyonuna yabancı değilim. İş tanımının gerektirdiği diğer gereksinimleri de en kısa zamanda öğrenmeye hazırım. Bu alanda ilerleyip bir gelişim kaydetme hedefindeyim.
